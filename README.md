# 📊 Online Retail Data Analysis Project

## 📌 Project Overview
This project analyzes an online retail dataset to uncover insights related to sales performance, customer behavior, product trends, and regional distribution.

The project is built using a complete data analysis workflow, covering:
- Data extraction and analysis (SQL)
- Data cleaning and exploration (Excel)
- Data visualization and storytelling (Tableau)

---

## 🛠️ Tools & Technologies Used

- SQL → Data querying and analysis  
- Excel → Data cleaning and preprocessing  
- Tableau → Interactive dashboard and visualization

## 📊 Key Analysis Areas

### 1. Sales Performance
- Revenue trends over time
- Order volume analysis
- Average Order Value (AOV)

### 2. Product Analysis
- Top-performing products by revenue
- High-volume vs high-value products
- Contribution distribution (Pareto effect)

### 3. Customer Behavior
- Purchase patterns
- Spending trends

### 4. Geographic Insights
- Country-wise revenue contribution
- Identification of high-performing regions

### 5. Order Status Analysis
- Delivered vs Cancelled orders
- Cancellation rate evaluation

---

## 📈 Key Insights

- Revenue shows a strong upward trend over time  
- A small number of products contribute significantly to total revenue  
- AOV remains relatively stable with slight growth in later months  
- Certain countries dominate sales performance  
- Cancellation rate (~16%) indicates potential operational improvement areas
